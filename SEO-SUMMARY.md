## Quick Start

### 1. Test Locally (30 seconds)
```bash
npm run start
```

Then visit:
- Sitemap: http://127.0.0.1:8787/sitemap.xml
- Robots: http://127.0.0.1:8787/robots.txt
- SEO Monitor: http://127.0.0.1:8787/backoffice.html (Section 14)

### 2. Update Settings (2 minutes)
Open Back Office > Settings and fill in:
- Business name, phone, email, address
- Social media URLs (Facebook, Instagram, LinkedIn)
- Logo URL

### 3. Verify Schema (1 minute)
1. Go to: https://search.google.com/test/rich-results
2. Enter: `https://elitequalityhvac.com`
3. Confirm "HVACBusiness" and "Organization" detected
