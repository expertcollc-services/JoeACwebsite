# SEO Quick Start Checklist

## 🚀 5-Minute Setup

### 1. Update Business Info (2 min)
Open Back Office > Settings and fill in:
- [ ] Business name
- [ ] Phone number  
- [ ] Email address
- [ ] Physical address
- [ ] Social media URLs

### 2. Verify Sitemap (30 sec)
Visit: `http://127.0.0.1:8787/sitemap.xml`
- [ ] Confirm 9+ URLs are listed
- [ ] Check last modified date is today

### 3. Check Schema (1 min)
1. Visit: https://search.google.com/test/rich-results
2. Enter: `https://elitequalityhvac.com`
3. [ ] Confirm "HVACBusiness" detected
4. [ ] Confirm "Organization" detected
5. [ ] Confirm no errors

### 4. Monitor Dashboard (1 min)
Open: `http://127.0.0.1:8787/backoffice.html`
Navigate to: **Section 14: SEO Monitor**
- [ ] Review schema status
- [ ] Check technical SEO items
- [ ] Review local SEO checklist

### 5. Create Images (30 sec)
Create and upload:
- [ ] `/og-image.jpg` (1200x630px)
- [ ] `/twitter-image.jpg` (1200x600px)
- [ ] `/logo.png` (180x60px)
